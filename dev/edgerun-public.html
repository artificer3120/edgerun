edgerun mesh

An interactive map of every device a node can see via passive RF and system enumeration. Layers are toggleable so you can flip wifi-neighbors on or off, etc. Drag a node to grab it; release to let it float again.

scan from your phone

Termux + termux-api on Android. Output goes to ~/storage/downloads/edgerun-mesh.json. Then tap load mesh.json in the sidebar and pick the file.

pkg install termux-api python
termux-setup-storage
curl -O https://neonforge.untitledprojects.io/media/edgerun-probe.sh
bash edgerun-probe.sh

You'll need the Termux:API companion app installed too (F-Droid). Mobile probes capture wifi APs around you. Bluetooth scan from Termux is limited; on iOS, no public scan API exists at all — viewer only.

scan from a desktop

The PowerShell probe (Windows) covers wifi, bluetooth (paired), LAN ARP table, USB, and tailnet peers. Reach out to the artificer if you want it.

ethics line. Passive RF observation in public is the same thing wifi-survey apps do. This tool does not aggregate, upload, or share scans — everything stays on your device. Don't pair MACs to people. Don't probe networks. Don't scan in private spaces where you wouldn't be welcome doing it openly. Listen, never poke.
